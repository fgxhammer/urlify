# URL shortener with :whale: Docker :seedling: Mongodb :vulcan_salute: Vue :rocket: Express

## Database
Make sure you have Docker installed - if not get it [here](https://www.docker.com/products/docker-desktop)
### Start Database:
```
cd database
docker-compose up
```
## Server
### Start server:

```
cd server
npm install
npm run dev
```
